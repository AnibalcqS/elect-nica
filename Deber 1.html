Sentencias de Control en Programación

1. ¿Qué son las Sentencias de Control?

Las sentencias de control permiten controlar el flujo de ejecución de un programa. Sin ellas, el código se ejecutaría de forma estrictamente secuencial. Gracias a estas estructuras, el software puede tomar decisiones y realizar tareas repetitivas de forma eficiente.

2. Sintaxis Fundamental

A. Condicional If / Else

Evalúa si una condición es verdadera para ejecutar un bloque de código.

if (condicion) {
    // código si es verdadero
} else {
    // código si es falso
}

B. Bucle For

Repite un bloque de código un número determinado de veces.

for (let i = 0; i < 5; i++) {
    // código que se repite 5 veces
}
