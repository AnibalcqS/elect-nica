Teoría de Funciones

1. ¿Qué es una Función?

Una función es un bloque de código reutilizable diseñado para realizar una tarea específica. En lugar de escribir el mismo código varias veces, lo encerramos en una función y lo "llamamos" cada vez que lo necesitamos.

Analogía: Imagina una función como una receta de cocina. La receta (función) te dice qué pasos seguir. Tú puedes usar esa receta (llamar a la función) cuantas veces quieras para obtener un pastel (resultado).

2. Tipos de Funciones y Sintaxis

A. Función Declarativa (Básica)

Es la forma más común de definir una función.

function saludar(nombre) {
    return "Hola " + nombre;
}

B. Funciones de Flecha (Arrow Functions)

Una sintaxis moderna y corta muy usada en JavaScript actual.

const sumar = (a, b) => a + b;
