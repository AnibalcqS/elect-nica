Consolidación de Conceptos de Expresiones, Operadores y Funciones

1. ¿Qué es una expresión?

Una expresión es cualquier unidad de código válida que se resuelve en un valor. En esencia, es una combinación de variables, constantes y operadores que el intérprete evalúa.

  • Expresiones Aritméticas: Evalúan a un número (ej: 10 + 5).
  • Expresiones Lógicas: Evalúan a un booleano (ej: x > y).
  • Expresiones de Cadena: Evalúan a texto (ej: "Hola " + nombre).

2. Tipos de Operadores

Los operadores permiten manipular los valores de las expresiones:

Tipo Operadores Descripción
Aritméticos +, -, *, /, % Para realizar cálculos matemáticos.
Relacionales >, <, >=, <=, === Comparan dos valores y devuelven True/False.
Lógicos &&, ||, ! Combinan múltiples condiciones booleanas.

3. 5 Ejercicios sobre Funciones

Código fuente de las soluciones paso a paso:

// 1. Función para duplicar un número
const duplicar = (n) => n * 2;

// 2. Función para saludar
const saludar = (nombre) => "Hola, " + nombre;

// 3. Función para validar mayoría de edad
const esAdulto = (edad) => edad >= 18;

// 4. Función para convertir Celsius a Fahrenheit
const convertirTemp = (c) => (c * 9/5) + 32;

// 5. Función para calcular área de un cuadrado
const areaCuadrado = (lado) => lado * lado;

5. Calculadora Básica (2 Puntos)

Código principal de la lógica de procesamiento (Visual Studio):

function resolver() {
    const screen = document.getElementById('calc-display');
    try {
        // eval() procesa la cadena aritmética (ej: "5+5*2")
        screen.value = eval(screen.value);
    } catch {
        screen.value = "Error";
    }
}
