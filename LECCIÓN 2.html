Documentación: Comandos de Figma

Guía profesional para el diseño de interfaces (UI/UX)

1. Selección y Manipulación

Comando: Move Tool

¿Qué hace? Selecciona, mueve y escala elementos en el lienzo.

Uso: Es la herramienta base para organizar capas.

Acceso rápido: V

Comando: Frame Tool

¿Qué hace? Crea contenedores (pantallas) con dimensiones predefinidas (iPhone, Desktop).

Uso: Para delimitar el área de diseño de cada vista.

Acceso rápido: F

2. Herramientas de Dibujo

Formas: Rectangle / Ellipse

¿Qué hace? Inserta geometrías básicas para botones o contenedores.

Acceso rápido: R o O

Pen Tool (Vectores)

¿Qué hace? Crea formas personalizadas e iconos mediante nodos.

Acceso rápido: P

3. Alineación y Distribución

Fundamental para mantener la limpieza visual y el orden en la interfaz.

Auto Layout: El comando más potente de Figma para crear diseños responsivos que se ajustan al contenido.

Acceso rápido: Shift + A

Alineación: Usar los iconos en la parte superior derecha para alinear al centro, izquierda o distribuir espacios iguales.

4. Componentes y Estilos

Create Component: Convierte un objeto en un "Maestro". Si cambias el maestro, todas las instancias cambian.

Acceso rápido: Ctrl + Alt + K

Color Styles: Guarda paletas de colores globales para asegurar la consistencia de marca.

5. Herramientas de Prototipado

Permite dar vida al diseño creando conexiones entre pantallas.

Interacciones: Define qué sucede al hacer "Click", "Hover" o "Drag".

Smart Animate: Figma detecta cambios en capas con el mismo nombre y crea una transición suave automáticamente.
