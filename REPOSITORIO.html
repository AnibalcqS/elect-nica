Convertidor de Sistemas Numéricos

Descripción

Aplicación en Python que convierte números entre diferentes sistemas numéricos: Binario, Octal, Decimal y Hexadecimal. Incluye interfaz gráfica con Tkinter y colores diferenciados para cada base.

Funcionalidades

  • Ingresar números en Binario, Octal, Decimal o Hexadecimal.
  • Convertir automáticamente a las otras tres bases.
  • Mostrar resultados con colores según la base.
  • Manejo de errores si el número ingresado no corresponde a la base seleccionada.

Tecnologías

  • Python 3.x
  • Tkinter (Interfaz gráfica)
  • Compatible con Windows, Linux y macOS

Instalación

  1. Instalar Python 3: https://www.python.org/downloads/
  2. Clonar o descargar este proyecto en tu computadora.
  3. Abrir el proyecto en Visual Studio Code.
  4. Ejecutar el archivo principal: 
    python convertidor.py

Uso

  1. Escribir un número en el campo de entrada.
  2. Seleccionar la base del número ingresado.
  3. Presionar Convertir.
  4. Ver los resultados en las otras tres bases con colores.



Ejemplo

Entrada: 1010

Base seleccionada: Binario

Resultados:

  • Decimal: 10
  • Octal: 12
  • Hexadecimal: A
